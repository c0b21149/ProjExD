## 避けろ！こうかとん（ex06/jump_koukaton.py）
### ゲーム概要
- ex06/jump_koukaton.pyを実行すると1600×900のスクリーンに草原が描写され
こうかとんをジャンプさせ迫りくる爆弾を避けるゲーム
### 操作方法
- 矢印キーの上でジャンプ下で着地までの時間を早くする。

- 着地が早くなるようにした
